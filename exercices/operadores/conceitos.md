# Operadores
## Operadores Aritmeticos
operador|significado
-----|-----
+|adição
-|subtração
*|multiplicação
/|divisão
//|divisão inteira
%|resto


**Exemplos**:
```python
3 + 2
3 - 2
3 * 2
3 / 2
3 // 2
3 % 2
```

## Operadores Lógicos

operador|significado
-----|-----
==|igualdade
!=|diferente
\>|maior
\>=|maior ou igual
<|menor
<=|menor ou igual

**Exemplos**:
```python
3==3
'a'=='a'
'a'!= 3
(3 * 2) > (3 + 2)
(3 * 2) >= (3 + 2)
(3 * 2) < (3 + 2)
(3 * 2) <= (3 + 2)

```
## Operadores de atribuição

operador|significado
-----|-----
+=| adiciona a variável
-=| subtrai da variável
*=| multiplica a variável
/=| divide a variável por
//=| divide a variável por
%=| atribui o resto da divisão
:=| cria e atribui valor

**Exemplos**:
```python
a = 1   # => 1
a += 4   # => 5
a -= 2   # => 3
a *= 7   # => 21
a /= 2   # => 10.5
a //= 3   # => 3.0
a %= 2   # => 1.0
a := 3 * 7
```
